📘 주가 예측 모델 설명

1. 사용 데이터

출처: Yahoo Finance (yfinance)
대상: 애플(AAPL) 주식 데이터
범위: 최근 5일간의 5분봉 데이터

2. 라벨링 방식

시가(Open)와 종가(Close)의 차이를 기준으로 다음과 같이 라벨링합니다:

  • 상승: 종가 - 시가 > 0.07
  • 하락: 종가 - 시가 < -0.07
  • 횡보: 나머지 구간
  • Threshold 수치 근거:0.05~0.08의 구간에서 각 구간의 분포 균형도를 표준편차로 비교하였을때, 0.07의 분표 균형도가 가장 균형있었고, 동시에 예측정확도도 1.00으로 유지되어 0.07을 Threshold로 채택

3. 특징(feature) 목록

  • Open, High, Low, Close, Volume
  • Range (고가 - 저가)
  • Body (시가-종가의 절댓값)
  • Direction (Close - Open)
  • Volatility ((High - Low) / Open)

4. 사용 모델

모델: XGBoost Classifier
주요 하이퍼파라미터: max_depth=4, learning_rate=0.1, n_estimators=100 모델 채택 사유:총 4개의 모델을 비교 분석하였을때(XGBoost, Random Forest, LogisticRegression, SVM) XGBOOST와 Random Forest의 정확도가 가장 높게 나왔고,
이중 Boosting 기반 주조로 더 정밀하게 오차 보정과 과적합 억제조절이 가능하여 더 실전성이 높다고 생각하여 XGBoost를 채택
